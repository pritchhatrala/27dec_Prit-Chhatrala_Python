26. String slicing.
String slicing allows you to extract a portion of a string using the syntax string[start:end:step]. The start index specifies where the slice begins (inclusive), the end index specifies where it stops (exclusive), and the step determines the increment. For example, "Python"[0:4] returns "Pyth" (characters from index 0 to 3), and "Python"[::2] returns "Pto" (every second character). Negative indexing allows slicing from the end, e.g., "Python"[-3:] returns "hon", and reversing a string can be done using "Python"[::-1], which gives "nohtyP". String slicing is a powerful way to manipulate substrings efficiently.
